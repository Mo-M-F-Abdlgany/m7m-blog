الخطوه الاولي : قم بتسجيل الدخول إلي حسابك علي جيت هب (github.com)

الخطوه الثانيه : قم بإنشاء مستودع جديد (Create new repository)

الخطوه الثالثه : قم بتسميه المستودع واضغط علي زر الانشاء(Create repository)

لخطوه الرابعه : قم بالضغط علي رابط رفع ملف موجود بالفعل (Uploading an existing file)

الخطوه الخامسه : 1-قم بسحب الملفات او قم بالضغط علي رابط اختيار ملفك شاهد كيف تسحب الملفات من هنا (Choose your files)
2- اضغط علي زر حفظ التغييرات (Commit changes )

هذا الجزء خاص ب Git hub live pages
* تنويه* يجب ان يكون اسم الصفحه الريئسيه (index)

الخطوه السادسه : قم بالذهاب للإعددات(Settings), وأذهب إلي قسم صفحات جيت هب (Git hup pages)

الخطوه السابعه : قم بتغيير الإعددات إلي الاعدادت الموضحه بالصوره ثم اضغط حفظ (Save)

الخطوه الثامنه: تهانينا لقد تم نشر موقعك قم بالضغط علي الرابط للذهاب إليه
